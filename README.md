# proyecto-clasificadorPiezas-deepLearning
Sistema de Visión Artificial para clasificar piezas industriales en tiempo real (1 m/s) con Deep Learning. Emplea MobileNetV2 sobre el dataset MVTec. Incluye dimensionamiento de hardware (cámara Global Shutter) y preprocesamiento avanzado (CLAHE, alineación geométrica y Data Augmentation). Alcanza un 96% de Accuracy y latencia mínima.
